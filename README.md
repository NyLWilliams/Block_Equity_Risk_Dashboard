# Block_Equity_Risk_Dashboard
Tableau dashboard analyzing document completeness and risk status for private equity operations.
## Key Business Questions
- Which portfolio companies are currently **high risk**?
- How many required documents are **missing vs uploaded** per entity?
- Which companies require **immediate follow-up**?

---

## Metrics Used
- **Total Required Documents**
- **Uploaded Required Documents**
- **Missing Required Documents**
- **Risk Status** (derived from document completeness)

---

## Risk Logic
- **High Risk** → One or more required documents missing
- *(Logic can be extended to Medium / Low Risk in future iterations)*
